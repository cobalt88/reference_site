something else that can be used is the "cite" tag that can be put into another tag
such as "blockquote" to add a link to where the quote was taken from.
the difference between this and an href is it is not visible in the main webpage,
but it can be seen inside the HTML document. It always required to cite any work,
be it code or text copied directly from another page. To fail to do so is Plagarism.

speaking of citing, you can also use the "cite" tag on its own to show the name

another text formatting feature is to make the text Strong. Strong is different from bold in that it is intended to be used for text that is more important than what is around it.
