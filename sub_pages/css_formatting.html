CSS Selectors
CSS selectors are used to target the HTML elements you want to style
Selector Description Example/Syntax
#id The ID (#) selector targets HTML elelments based on their assighned id's, for exmaple <div id="target-me"> is a div element with the ID of #target-me

#target-me {
add anything you like in here. like background-color: white;
}
.class The class (.class) selector is likely the one you will use the most. You can assign a class to an elements by using class="class-name-here" inine of the HTML tag. .class-name-here {
CSS attributes here
}
element.class This selector targets elements within classes. For example if you have a paragraph <p> nested within the <div class="class-name-here"> you can select it with p.class-name-here p.class-name-here {
any CSS attributes you place here will will target all <p> elements that are children of any parent elements of the .class-name-here class.
}
