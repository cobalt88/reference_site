Note: by default a hyperlink will open in the tab you are currently in unless you add target="blank" after href="". adding a blank target will cause the hyperlink to open in a new tab.

For general security reasons it is best to add rel="noopener noreferrer" if your going to have a blank target. The sytax for a hyperlink that openes in a new tab or window (in this example I am using www.google.com as the page the hyperlink will open) is as follows:

<a href="www.google.com" target="blank" rel="noopener noreferrer">

<head> The <head> tag is not to be confused with the <header> tag, they both serve a different purpose. <head> tags are used to define the head of an HTML document, this is where metadata and other information such as links to external css and JS pages that apply to the entire document can be found.
Note: Each HTML document only has one <head> section. 		For an example of this tags syntax and use view the page source and look at the very top of the page. This item is also explained in more detail in the HTML Basics page.
<header> The <header> tag is used to define the Header for a document or section. Unlike the <head> tag, you can have multiple headers, just like you can have multiple bodies and footers. This will be desplayed in more detail in the HTML Basics 2 Page.
