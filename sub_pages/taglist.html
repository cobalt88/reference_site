<main> This tag is a great one to know, It specifies the main content of a document. But make sure you understand how to use it, there is a list of conditions for its use.
  1. There can only be 1 <main> element in a document. unlike most other HTML elements you can use this one as many times as you want to describe differrent sections of your content, thats what <section> is for.

  2. This element can only be used to contain content unique to the document it is in. So things like navigation bars, contact info, copyright information, anything else that is used across multiple pages on your site, CANNOT be placed inside this element

  3. and lastly <main> MUST NOT be a child element or decendant of an <article>, <aside>, <footer>, <header>, or <nav> element.
